| Action       | URL                | Method | Body / Header Needed?       | Result Example       |
| ------------ | ------------------ | ------ | --------------------------- | -------------------- |
| Get Events   | /api/events        | GET    | none                        | JSON array of events |
| Register     | /api/auth/register | POST   | JSON body                   | JSON of user         |
| Login        | /api/auth/login    | POST   | JSON body                   | JWT token            |
| Create Event | /api/events        | POST   | JSON + Authorization header | JSON of new event    |
| Book Event   | /api/bookings      | POST   | JSON + Authorization header | JSON of booking      |
